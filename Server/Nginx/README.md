## Nginx

### nginx安装
  - 修改yum源
  ```
  >cat /etc/yum.repos.d/nginx.repo
  [nginx]
  name=nginx repo
  baseurl=http://nginx.org/packages/centos/7/$basearch/
  gpgcheck=0
  enable=1
  ```
  - yum install nginx
